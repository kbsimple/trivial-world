## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Touch targets: 44px minimum (iOS HIG) — applies to all interactive elements
- Category badge padding: 16px horizontal / 8px vertical (per existing CategoryBadge component)

## Typography

Established from existing components (QuestionCard, CategoryBadge, screens):

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label | 14px | 600 (semibold) | 1.4 |
| Heading | 28px | 700 (bold) | 1.2 |
| Display | 32px | 700 (bold) | 1.1 |

**Font sizes in use:**
- Question text: 24px bold (eyes-up design for conductor readability)
- Button text: 18px semibold/bold
- Category badge: 16px bold (size `$3` default)
- Hint text: 12px regular (opacity 0.7)

## Color

Per existing Tamagui config and CATEGORY_COLORS constant:

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#1a1a2e` | Background (dark theme) |
| Secondary (30%) | `rgba(255,255,255,0.1)` | Cards, rows, input backgrounds |
| Accent (10%) | `#228b22` | Primary action buttons (Start Game, Download Pack) |
| Destructive | `#ff6b6b` | Remove/delete actions, error states |

**Category colors (reserved for category badges and indicators):**
- Blue: `#0066cc` — The World Outside
- Pink: `#ff69b4` — Pop Culture & Streaming
- Yellow: `#ffd700` — Milestones & Myths
- Purple: `#9932cc` — Animation and Artwork
- Green: `#228b22` — Tech, Space & Logic
- Orange: `#ff8c00` — Sports & Gaming

**Accent reserved for:**
- Primary CTA buttons (Start Game, Download Pack, Confirm Selection)
- Active/selected state indicators
- Success confirmations

## Component Patterns

### Pack List Item

```
┌─────────────────────────────────────────────────────────────┐
│ [Pack Name]                           [Update Available]    │
│ by {Author} • {totalQuestions} questions                    │
│                                        [Download] / [Select] │
└─────────────────────────────────────────────────────────────┘
```

States: `available` (not downloaded), `downloaded`, `update-available`, `active`

### Pack Details Modal

```
┌─────────────────────────────────────────────────────────────┐
│                    {Pack Name}                      [Close] │
├─────────────────────────────────────────────────────────────┤
│ by {Author} • v{version}                                    │
│ {description}                                               │
│                                                             │
│ Category Distribution:                                      │
│ ━━━━━━━━━━ Blue (20)                                        │
│ ━━━━━━━━━━ Pink (20)                                        │
│ ━━━━━━━━━━ Yellow (20)                                      │
│ ━━━━━━━━━━ Purple (20)                                      │
│ ━━━━━━━━━━ Green (20)                                       │
│ ━━━━━━━━━━ Orange (20)                                      │
│                                                             │
│ Difficulty: Easy (40) • Medium (50) • Hard (30)             │

│                                                             │

```

### Category Filter Toggles

```
┌─────────────────────────────────────────────────────────────┐
│ Select Categories                                           │
│                                                             │
│ [●] Blue — The World Outside                                │
│ [●] Pink — Pop Culture & Streaming                          │
│ [●] Yellow — Milestones & Myths                             │
│ [●] Purple — Animation and Artwork                          │
│ [●] Green — Tech, Space & Logic                            │
│ [●] Orange — Sports & Gaming                               │
│                                                             │
│ [Select All] [Clear All]                                    │
└─────────────────────────────────────────────────────────────┘
```

- Toggle state: filled circle (●) for enabled, empty circle (○) for disabled
- Category names use CATEGORY_NAMES constant
- Color indicators use CATEGORY_COLORS background

### Download Progress Bar

```
┌─────────────────────────────────────────────────────────────┐
│ Downloading {packName}...                                   │
│ ████████████░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
│ 35% • 15.2 MB / 43.8 MB                                    │
└─────────────────────────────────────────────────────────────┘
```

- Progress bar: green fill (`#228b22`), rounded corners
- Show percentage and bytes transferred
- Non-cancelable (user must wait for completion or error)

### Difficulty Filter

```
┌─────────────────────────────────────────────────────────────┐
│ Difficulty                                                  │
│ [●] Easy    [●] Medium    [●] Hard                         │
│ (All selected by default)                                   │
└─────────────────────────────────────────────────────────────┘
```

## Interaction States

### Pack List Item

| State | Visual |
|-------|--------|
| Default | White/light text on transparent background |
| Pressed | Opacity 0.7 |
| Downloaded | "Downloaded" label, "Select" button visible |
| Update Available | Yellow/amber badge, "Update" button |
| Active (selected) | Green border, checkmark icon |

### Button States

| State | Visual |
|-------|--------|
| Default | Green background (`#228b22`), white text |
| Pressed | Darker green (`#1a6b1a`) |
| Disabled | 50% opacity, gray tint |
| Loading | Spinner icon replaces text |

### Category Toggle

| State | Visual |
|-------|--------|
| Enabled | Filled circle, category color background |
| Disabled | Empty circle, gray border |
| Pressed | Scale down 0.95 briefly |

## Navigation Flow

Per D-01: Pack selection screen BEFORE setup screen.

```
Home Screen
    │
    ▼
Pack Selection Screen (NEW)
    │ ├── Pack List (downloaded + available)
    │ ├── Pack Details Modal (per pack)
    │ └── Category/Difficulty Filters
    │
    ▼
Setup Screen (existing)
    │ └── Receives selected pack ID
    │
    ▼
Game screens (existing)
```

**Back navigation:**
- Pack Selection → Home (back button)
- Pack Details Modal → Pack Selection (close button)
- Setup → Pack Selection (if no pack selected)

## Accessibility

- All touch targets: minimum 44px height
- Color contrast: WCAG 2.1 AA minimum (4.5:1 for text)
- Category toggles: include `accessibilityLabel` and `accessibilityRole="checkbox"`
- Download progress: `accessibilityLabel` updates with percentage
- Error states: `accessibilityHint` describes recovery action
- Pack details modal: focus trap, escape key closes (Android back gesture)
