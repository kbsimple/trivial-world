<objective>
Introduce per-player pack customization mode to the game setup screen.

Purpose: Replaces the existing implicit small per-player pack chips with a top-level toggle between "Shared Pack" (all players use game-level source) and "Per Player" (each player picks their own source). This is a purely additive change — no existing behavior is removed, the default is "Shared Pack" which matches current behavior.

Output:
- `packStore.ts`: `packMode: 'shared' | 'custom'` field + `setPackMode` action + `packMode` in partialize
- `packStore.test.ts`: two new tests for `setPackMode` initial state and action behavior
- `setup.tsx`: segmented control below pack banner, conditional per-player rows (shared=hidden, custom=full-width source row + difficulty chip)
</objective>
