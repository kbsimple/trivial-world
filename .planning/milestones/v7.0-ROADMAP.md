# Milestone v7.0: Per-Player Pack Customization

**Status:** ✅ SHIPPED 2026-06-13
**Phases:** 19
**Total Plans:** 1

## Overview

Introduced a top-level "Shared Pack" vs "Custom Per Player" toggle on the game setup screen. When "Custom Per Player" is selected, each player gets a dedicated full-width pack/combo selector. When "Shared" is selected, no per-player pack UI is shown. Switching from Custom to Shared automatically clears all player pack/combo overrides. The mode persists across app restarts via Zustand persist.

## Phases

### Phase 19: Per-Player Pack Customization

**Goal:** Introduce a top-level game option — "Shared Pack" or "Custom Per Player" — that controls whether all players draw from the same pool or each player picks their own. When "Custom Per Player" is selected, each player gets a dedicated pack/combo selector in the setup screen. When "Shared" is selected, no per-player pack UI is shown. This replaces the existing small per-player chips with a clearer, intentional flow.
